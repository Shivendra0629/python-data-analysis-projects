# Student Performance Visualization

## Project Overview

This project analyzes student academic data and visualizes performance trends using Python data visualization libraries.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## Features

* Analysis of student marks dataset
* Visualization of subject-wise performance
* Graphical representation of academic insights

## Visualizations

The project includes charts such as:

* Marks distribution graphs
* Subject-wise performance charts
* Overall academic trends

## How to Run

1. Download the project
2. Open the Jupyter Notebook file
3. Run all cells to generate visualizations
